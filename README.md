# Unvanquished stacking mod

Modifications sur cgame et sgame pour ajouter/modifier le comportement de placement et d'affichage des buildables.

## Fichiers inclus

- src/cgame/cg_buildable.cpp
- src/cgame/cg_draw.cpp
- src/cgame/cg_local.h
- src/sgame/sg_buildable.cpp
- src/sgame/backend/

## But

Ce dépôt contient uniquement mes modifications sources liées au mod, pas l'intégralité du projet Unvanquished.

## Base

À réintégrer sur une base compatible d'Unvanquished.
